# Quick knowledge React

React é uma biblioteca pra contrução de interfaces de usuários

Exemplo de renderização de componente react dentro do HTML:

```js
ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('root')
);
```
